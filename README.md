# Automated Electricity Supplier Price Optimizer

AI-powered platform for electricity price monitoring, forecasting, and automated supplier switching.

## Overview

This project implements a comprehensive solution for:
- Real-time electricity price monitoring across multiple suppliers
- ML-based price forecasting (CNN-LSTM model)
- Automated load shifting optimization (MILP)
- Intelligent supplier switching recommendations
- GDPR-compliant data management

## Tech Stack

**Backend**: FastAPI, Supabase (PostgreSQL), TimescaleDB, Redis, Celery
**ML/Data**: TensorFlow, XGBoost, Airflow, PuLP
**Frontend**: Next.js 14, React 18, Recharts, D3.js, Tailwind CSS
**Infrastructure**: Docker, Prometheus, Grafana, GitHub Actions
